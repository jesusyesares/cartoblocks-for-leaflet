# Palabras y frases a evitar

Lista de palabras, expresiones y construcciones sobreutilizadas por la IA en español de España. Cada entrada incluye alternativas naturales.

No se trata de prohibir palabras de forma absoluta. Si una palabra de la lista es la más precisa en un contexto concreto, puede usarse. Lo que se evita es el uso recurrente, automático y desproporcionado que delata texto generado por máquina.

## Palabras de prohibición absoluta

Estas palabras o expresiones no deben usarse nunca. Están tan asociadas al texto IA o resultan tan artificiales en español natural que siempre hay una alternativa mejor.

| Prohibida | Alternativas |
|-----------|-------------|
| iterar, iteración | repetir, ajustar, mejorar, dar otra vuelta |
| fricción | problema, obstáculo, resistencia, lío |
| granular, granularidad | detallado, en detalle, al milímetro, con lupa |
| epítome | ejemplo perfecto, la definición misma de |
| inquebrantable | firme, constante, que no se rinde |
| vibrante | animado, vivo, con vidilla |
| crucial | importante, clave, que marca la diferencia |
| indeleble | que no se borra, permanente, que deja huella |
| desbloquear | conseguir, abrir, activar, lograr |
| por qué importan (como título) | reescribir como afirmación directa |

## Aperturas predecibles

Frases típicas de IA para empezar párrafos o artículos. Elimínalas y empieza directamente por el contenido.

| Evitar | Qué hacer |
|--------|-----------|
| En el mundo actual... | Empezar por el dato o la idea concreta |
| En un mundo cada vez más... | Ir directo al tema |
| Cabe destacar que... | Decirlo sin más, sin anunciar que lo vas a decir |
| Es importante señalar que... | Señalarlo directamente |
| No es ningún secreto que... | Afirmar sin preámbulo |
| Como todos sabemos... | Eliminar, es condescendiente |
| Hoy en día... | Empezar por lo que quieres contar |
| En este contexto... | Dar el contexto de forma natural |
| A lo largo de los años... | Ser específico: "Desde 2018..." |
| Vamos a ver... / Veamos... | Empezar directamente |
| Lo cierto es que... | Decirlo sin rodeos |
| No cabe duda de que... | Afirmar directamente |
| Resulta evidente que... | Decirlo y punto |
| Antes de nada... | Empezar por lo primero |

## Cierres mecánicos

Frases que la IA pone al final de secciones o artículos como resumen innecesario.

| Evitar | Qué hacer |
|--------|-----------|
| En definitiva... | Eliminar y dejar la última idea como cierre |
| En resumen... | Solo en textos muy largos donde realmente haga falta |
| En conclusión... | Cerrar con la idea final sin anunciarlo |
| Como hemos visto... | Eliminar, el lector ya lo ha leído |
| Para terminar... | Terminar sin avisar |
| Dicho esto... | Eliminar o usar "Eso sí" |
| En última instancia... | Eliminar |
| Todo esto nos lleva a... | Eliminar |

## Conectores formales excesivos

En español natural se usan conectores, pero la IA abusa de los más formales y académicos.

| Sobreutilizado | Alternativas naturales |
|----------------|----------------------|
| Asimismo | además, también, y (a veces basta con enlazar frases) |
| No obstante | pero, aunque, eso sí |
| Por consiguiente | así que, por eso, entonces |
| En este sentido | (eliminar, suele sobrar) |
| De esta manera | así |
| Con respecto a | sobre, en cuanto a |
| Cabe mencionar | (eliminar y mencionarlo directamente) |
| Es menester | hay que, hace falta, toca |
| En virtud de | por, gracias a |
| A tenor de | según |
| Aunado a | junto con, además de |
| En aras de | para, con el fin de |
| De cara a | para, pensando en |

## Palabras infladas

Vocabulario que la IA usa para dar un tono de importancia artificial. Casi siempre hay una alternativa más sencilla.

| Inflada | Alternativas |
|---------|-------------|
| panorama | situación, escenario, lo que hay |
| ecosistema | entorno, mundo, sector |
| paradigma | modelo, enfoque, forma de hacer las cosas |
| sinergia | colaboración, trabajo conjunto, combinación |
| catalizador | impulso, detonante, lo que lo disparó |
| implementar | poner en marcha, aplicar, montar |
| optimizar | mejorar, afinar, ajustar |
| potenciar | mejorar, reforzar, darle un empujón |
| fomentar | promover, animar, empujar |
| abordar | tratar, afrontar, ocuparse de |
| brindar | dar, ofrecer |
| garantizar | asegurar (con matiz: a veces ningún sinónimo es exacto) |
| innovador | nuevo, original, diferente |
| disruptivo | rompedor, que cambia las reglas (mejor: describir qué cambia) |
| holístico | completo, global, que lo cubre todo |
| robusto | sólido, fuerte, fiable |
| escalable | que aguanta el crecimiento, que se puede ampliar |
| apalancarse | apoyarse, sacar partido |
| empoderar | dar herramientas, capacitar, dar poder |
| impactar (como verbo transitivo) | afectar, influir en, repercutir |
| visibilizar | hacer visible, mostrar, sacar a la luz |
| transversal | que afecta a todo, común a todos |
| vertebrar | organizar, estructurar, articular |
| pivotar | cambiar de dirección, reorientar |
| monetizar | ganar dinero con, rentabilizar |
| dinamizar | animar, activar, darle movimiento |

## Metáforas gastadas

La IA repite estas metáforas de forma compulsiva. Sustitúyelas por descripciones concretas.

| Gastada | Qué hacer |
|---------|-----------|
| pilar fundamental | base, lo más importante, lo que sostiene |
| piedra angular | base, fundamento, lo esencial |
| motor de cambio | lo que empuja el cambio, lo que lo provoca |
| tejido social | sociedad, comunidad, relaciones entre personas |
| hoja de ruta | plan, pasos a seguir |
| punta de lanza | lo más avanzado, quien va primero |
| poner en valor | destacar, reconocer |
| a la vanguardia | por delante, de los más avanzados |
| punto de inflexión | momento de cambio, cuando todo cambió |
| un antes y un después | un cambio importante |
| abre las puertas a | permite, facilita, hace posible |
| sienta las bases | prepara el terreno, establece los cimientos |
| romper moldes | hacer algo diferente |
| cambio de paradigma | cambio de enfoque (o describir qué cambia exactamente) |
| una sinfonía de | (eliminar, describir directamente) |
| un tapiz de | (eliminar, describir directamente) |
| un mosaico de | (eliminar, describir directamente) |

## Inflación de importancia

Adjetivos y expresiones que la IA usa para dar un peso desproporcionado a cualquier tema.

| Inflada | Alternativas |
|---------|-------------|
| transformador | que cambia cosas, que mejora, importante |
| revolucionario | nuevo, diferente, que rompe con lo anterior |
| sin precedentes | nuevo, nunca visto (solo si es literalmente cierto) |
| referente | ejemplo, modelo |
| pionero | de los primeros, que abrió camino |
| extraordinario | muy bueno, notable, destacado |
| notable | bueno, destacado, que llama la atención |
| significativo | importante, considerable |
| trascendental | importante (si realmente lo es) |
| emblemático | conocido, representativo |

## Muletillas de énfasis

Expresiones que la IA añade para dar peso pero que no aportan información.

| Evitar | Qué hacer |
|--------|-----------|
| sin duda | eliminar (si no hay duda, no hace falta decirlo) |
| resulta fundamental | es importante, es clave |
| juega un papel clave/fundamental | es importante para, influye en |
| se ha convertido en un referente | es un buen ejemplo, destaca |
| merece la pena destacar | (eliminar y destacar directamente) |
| no podemos olvidar que | (decirlo sin más) |
| es necesario recalcar | (recalcarlo directamente) |
| llama poderosamente la atención | destaca, sorprende |
| cobra especial relevancia | es especialmente importante |
| resulta especialmente interesante | es interesante |

## Adverbios sobreutilizados

La IA abusa de adverbios que aportan poco.

| Evitar | Qué hacer |
|--------|-----------|
| significativamente | mucho, bastante, de forma notable |
| considerablemente | mucho, bastante |
| indudablemente | (eliminar, afirmar directamente) |
| ciertamente | (eliminar o usar "sí que") |
| notablemente | (eliminar o ser específico) |
| fundamentalmente | sobre todo, principalmente |
| esencialmente | en el fondo, básicamente |
| particularmente | sobre todo, especialmente |
| genuinamente | de verdad, de forma auténtica |
| inherentemente | por naturaleza, de por sí |

## Verbos comodín

| Comodín | Alternativas según contexto |
|---------|----------------------------|
| explorar | investigar, probar, analizar, estudiar, mirar |
| profundizar | entrar en detalle, analizar a fondo |
| navegar (por dificultades) | afrontar, lidiar con, manejar |
| aprovechar | usar, sacar partido, utilizar |
| facilitar | permitir, ayudar a, hacer más fácil |
| impulsar | promover, mover, empujar |
| articular | organizar, expresar, plantear |
| canalizar | dirigir, orientar |
| materializar | hacer realidad, conseguir, lograr |
| conceptualizar | pensar, diseñar, plantear |
